关于健身舱的思考

2017-11-05

整理一下个人最近的思考，当做毛砖先抛一下，大家讨论讨论

关于总部一号的健身舱

我认为总部一号的健身舱的意义在于

  1. 证明我们有快速执行能力（计划->设计->实施->投放）
    • 从8月底的想法，到10月底的落地对外营运，只用了2个月的时间
  2. 利于对外合作的现场demo
    • 合作方对于实际舱体的认识相比于纸面上的设计图更加直观，不必想象，所见所得
    • 政府及相关领导喜欢看实际的东西，有没有用再说
    • 方便申请相关高科技企业资质
    • 方便申请相关产业基金
  3. 弱电，强电，装修环节等目前团队的相对弱项的经过了充分的预演，踩过了坑，之后铺开会更有经验
  4. 达到了试错的目的，就是在园区并不适合放置（原因后面会提到）
  5. 接触到了真正意义上的第一批用户（或者说潜在用户更为精确），了解了不同用户的对于健身锻炼的不同需求
    • 小白用户想什么？
    • 有过健身经验用户想什么？
    • 价格定位是否合适？
  6. 了解了真正投入运营可能会碰到的问题
    1. 空调是否制冷，能够迅速把空间的温度降到舒适
    2. 器械的味道多久可以散去
    3. 器械的使用方法如何介绍给用户
    4. 舱体在跑步时候的减震问题
    5. 部分标识和提示是否清晰

为什么说园区放置共享健身舱不合适？原因如下

  1. 一些居住较远的职工，下班就想急忙回家，或者参加其他的社交活动，对于继续在办公场所继续逗留有主观抗拒情绪
  2. 没有洗澡措施，需要在每天上班途中携带更换衣物，太麻烦
  3. 除了部分企业主和高管，园区大部分人收入不高，温饱，租房成本重压下，无法顾及健身需求，每周踩个mobike就认为已经在锻炼了，可以了

适合投放的区域

那么综上所述，适合投放的区域应该具备什么样的条件呢？

  1. 整体收入门槛，我初步计算应该是平均工作人口的收入要大于25万一年
  2. 平时的生活方式不太健康，比如久坐，吃喝比较多，35岁以上人群比较多
  3. 回家洗漱便利
  4. 租金成本低廉
  5. 周边1公里内没有健身工作室和健身会所
  6. 物业和居民高度配合

所以成熟的居住小区，中大型社区内部，为上上选，小区周边配套商业区为中上选，工作区域为下选

目前可能的战略方向选择

既然目前总部一号的健身舱展示意义大于实际运营意义，那么我们就可以不在总部一号的收入和运营指标上做文章，诸如Alisha要求的做1个月看看的，可以预计做不出什么数字，也无法通过在此地运营回本。我们要把我们在总部一号投放的目的明确告诉对方。

考虑到目前共享健身舱已经不是一个尚在准备阶段的停留在想法的概念，而是全国至少有4家拿了几千万的投资准备大干一场的竞争对手，每个都有至少二三十号全职员工在每天工作12小时全力冲刺。为了要在这场战斗中取得先机，我提供几个战略方向供参考：

  1. 在1-2个月内拿到第一笔千万级人民币投资，迅速和一些地产和小区物业谈下框架协议，抢占位置，同时制造和研发第一批10个以上的健身舱进行实地运营，成为跑道上的一员

    ​

  2. 采用加盟方式，我们出后台，技术，实现，加盟方负责落地和采购硬件，双方运营分成

    ​

  3. 和某个已经拿到投资的跑道选手合作，提供多人健身舱的技术和后台，以入股的方式进行合作运营

    ​

  4. 另辟蹊径，从目前的健身角度延伸其他的集装箱应用，重开一片智能场景

上面4个方式当然1号是我们的初衷，但是也是时间要求最紧迫的。
